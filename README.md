# Indian Agriculture Data Analysis

## Project Overview

This project presents an end-to-end data analysis of Indian agriculture using **Excel, MySQL, Python, and Tableau**. The objective is to analyze agricultural productivity, cultivation costs, and yield efficiency across different crops and Indian states through data cleaning, exploratory analysis, SQL queries, and interactive dashboards.

## Objectives

* Clean and prepare agricultural data.
* Analyze crop yield and cultivation costs.
* Compare agricultural performance across Indian states.
* Calculate yield efficiency.
* Build an interactive Tableau dashboard for data visualization.

## Tools & Technologies

* **Excel** – Data Cleaning
* **MySQL** – Data Querying & Analysis
* **Python** – Data Analysis and Visualization
* **Tableau Public** – Interactive Dashboard

## Project Workflow

1. Data Collection
2. Data Cleaning (Excel)
3. SQL Analysis (MySQL)
4. Exploratory Data Analysis (Python)
5. Dashboard Development (Tableau)
